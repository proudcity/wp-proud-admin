== Description ==
This plugin turns a Wordpress admin theme into a nice looking dashboard based on the Flat UI kit by [Design Modo](http://designmodo.com/flat "Design Modo") and provides a customizable login logo and a better looking login page.

It is completely Open Source and still in early stages of development so feel free to contribute to the code on [Github](https://github.com/nickhargreaves/WP_FlatAdmin/ "WP FlatAdmin on Github")

You can also contribute by reporting issues and making feature requests [here](http://github.com/nickhargreaves/WP_FlatAdmin/issues, "Report issues") 

== Installation ==

How to install the plugin and get it working.

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. To use your own logo go to the admin menu > Settings > WP Flat Admin. You'll be able to upload a logo or add a url. You can also change the link of the logo and the title.
